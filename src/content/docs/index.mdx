## About AIPs

Aptos Improvement Proposals (AIP) describe standards for the Aptos Network including the core blockchain protocol and the development platform (Move), smart contracts and systems for smart contract verification, standards for the deployment and operation of the Aptos Network, APIs for accessing the Aptos Network and processing information from the Aptos Network.

## AIP Types

- **Standard**: AIPs focusing on changes to the Aptos blockchain
- **Informational**: AIPs providing additional information, context, and supporting details

## AIP Statuses

| Status | Description |
|:--|:--|
| `Draft` | Drafts are currently in process and not ready for review |
| `In Review` | AIPs are ready for community review and feedback |
| `Accepted` | AIPs have been accepted and will be implemented soon |
| `Rejected` | A community decision has been made to not move forward at this time |
| `On Hold` | Some information is missing or prerequisites have not yet been completed |

## Categories

AIPs are organized into the following categories:

- **Framework**: Changes to the Move framework
- **Core**: Changes to the core blockchain protocol
- **Gas**: Gas-related improvements
- **Cryptography**: Cryptographic primitives and protocols
- **Ecosystem**: Ecosystem standards and tooling
- **Informational**: Educational and informational content
- **Governance**: Governance-related proposals
- **VM**: Virtual machine improvements
- **Networking**: Network protocol changes
- **Language**: Move language improvements
